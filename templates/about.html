About Our Project

Welcome to our eCommerce platform, a project designed to provide personalized product recommendations using advanced content-based filtering techniques. This platform is created as part of a learning experience to combine modern web technologies with effective recommendation systems.

  • Rojan Ghimire - Backend Developer.
  • Abhishek Kuikel: Guiding and mentoring the team to achieve project goals effectively.

Technology Stack

Our project leverages the following technologies:

  • Backend: Flask - A lightweight web framework for handling server-side logic.
  • Frontend: HTML, CSS, JavaScript - For creating a user-friendly and responsive interface.
  • Recommendation System: Content-based filtering implemented using Python and TF-IDF vectorization.

Project Goals

The primary goal of this project is to create an eCommerce platform that provides users with accurate and personalized product recommendations based on the content they interact with. The platform aims to deliver a seamless shopping experience while showcasing the practical applications of content-based filtering in a real-world scenario.
